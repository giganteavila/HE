Una vez realizada la tarea, el envío se realizará a través de la plataforma moodle.

- El formato del archivo será **Markdown** (`.md`)
- Si se envían más de un fichero, se empaquetarán en un solo fichero `.zip`.
- El archivo se nombrará siguiendo estas pautas:
	- **Apellido1_Apellido2_Nombre_HE01_Tarea**
		- Asegúrate de que el nombre **no contenga la letra "ñ", tildes ni caracteres especiales**. Por ejemplo, la alumna Begoña Sánchez Mañas, para la primera unidad del módulo de HE, debería nombrar esta tarea como:
```
			sanchez_manas_begona_HE01_Tarea.md
```
